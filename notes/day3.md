---

## 理解したこと

### APIとは

ユーザーからのリクエスト（URLやデータ）を受け取り、Pythonで処理して結果を返す仕組み

---

### GETとPOSTの違い

* GET：URLでデータを送る（取得）
* POST：JSONでデータを送る（登録・送信）

---

### パスパラメータ

* URLの中に埋め込まれる値
* 例：`/items/1` → item_id = 1

---

### クエリパラメータ

* URLの「?」の後ろに付くデータ
* 例：`/search?keyword=python&limit=5`

---

### デフォルト値

* `limit: int = 10` のように書くと、値が送られなかった場合に使われる

---

### JSON

* データの形式（キーと値のセット）
* 例：

```json
{
  "name": "apple",
  "price": 120
}
```

---

### POSTの処理

* JSON → Pythonの変数に変換される
* 例：

```python
item.name  → "apple"
item.price → 120
```

---

## つまずいたこと

* GETとPOSTの違いが最初分からなかった
* クエリパラメータの意味が曖昧だった
* JSONの役割が分からなかった

---

## 学び

* コードを書くだけでは理解できない
* 実際にURLやdocsで確認することで理解が進んだ
* 「入力 → 処理 → 出力」を意識すると分かりやすい
